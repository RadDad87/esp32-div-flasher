Which board do I have?

Show diagrams

Don't worry about picking wrong — the flasher auto-detects your chip. But here's how to tell them apart:

ESP32-DIV v1

ESP32 · Micro USB
  • Sandwich-style PCB (main + shield)
  • BOOT btn on ESP32 DevKit module
  • Micro USB on DevKit (bottom board)
  • Pin headers connect boards
  • 3x NRF24 + CC1101 on shield

ESP32-DIV v2

ESP32-S3 · USB-C
  • Single integrated PCB + shield via pogo pins
  • BOOT btn on left edge of main board
  • USB-C port on bottom edge
  • IP5306 battery management built-in
  • Soft tactile nav buttons
BOOT button (hold to enter flash mode)
USB port (connect here to flash)
RST / Antenna / Power

How to Flash

  1. Connect your ESP32-DIV via USB
  2. Click Flash on the firmware you want
  3. Select the COM/serial port in the popup
  4. Wait for the flash to complete (~60 seconds)
Auto-detect: The flasher reads your chip ID on connect and picks the correct build for DIV v1 (ESP32) or DIV v2 (ESP32-S3). No manual selection needed.
If the device isn't detected, hold the BOOT button while plugging in USB, then try again. See diagrams above for button location.
