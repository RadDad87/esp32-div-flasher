// Don't worry — the flasher auto-detects your chip. But here's how to tell them apart:

ESP32-DIV v1

ESP32 · Micro USB
  • Sandwich-style PCB (main + shield)
  • BOOT btn on ESP32 DevKit module
  • Micro USB on DevKit (bottom board)
  • Pin headers connect boards
  • 3x NRF24 + CC1101 on shield

ESP32-DIV v2

ESP32-S3 · USB-C
  • Single integrated PCB + shield via pogo pins
  • BOOT btn on left edge of main board
  • USB-C port on bottom edge
  • IP5306 battery management built-in
  • Soft tactile nav buttons
BOOT (hold to enter flash mode)
USB port (connect here)
RST / Antenna / Power
// how to flash
flash_guide.sh
$ connect # plug in your ESP32-DIV via USB
$ select_firmware # click the Flash button above
$ pick_port # choose the COM/serial port in the popup
$ wait --timeout 60s # flashing takes ~60 seconds
$ done # your DIV is ready
> auto-detect: the flasher reads your chip ID on connect and picks the correct build for DIV v1 (ESP32) or DIV v2 (ESP32-S3). No manual selection needed.
> troubleshoot: if the device isn't detected, hold the BOOT button while plugging in USB, then try again. See diagrams above for button location.
